D3.js Treemap

This D3.js Treemap is part of D3.js-Charts Github project. This documentation contains implementation of Treemap (interactive data visualization) using D3.js library . D3.js is a JavaScript library for manipulating documents based on data. D3 helps you bring data to life using HTML, SVG, and CSS.

The Code Pen of chart can also be accessed and forked at: Treemap

The Data Source used for the chart can be accessed at: Dataset

script.babel

Treemap implementation for comparing total revenue of movies grouped together by genres like action,drama,comedy etc and the size of each movie rectangle will be propotional to the revenue it generated. The data is grouped genre wise and the sizes are represented revenue wise using D3.js library.
