D3.js Bar Chart

This D3.js Bar Chart is part of D3.js-Charts Github project. This documentation contains implementation of bar chart(interactive data visualization) using D3.js library .D3.js is a JavaScript library for manipulating documents based on data. D3 helps you bring data to life using HTML, SVG, and CSS.

The Code Pen of chart can also be accessed and forked at: Bar Chart
